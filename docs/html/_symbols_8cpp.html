◆ overloadsTable

std::unordered_map< std::string, std::vector<char> > overloadsTable

A table to record all the overloads of all defined language symbols. The vector<char> represents a set of overloads of a given basename. Values >=0 are for user symbols, the value itself gives the number of arguments. -1 is for n-matched symbols. Values <=-2 are for built-in symbols, -(value+2) gives the number of arguments for a built-in symbol.

◆ symbolTable

std::unordered_map< std::string, SymbolTableElement > symbolTable

Records the values of all symbols. Keys have the form basename?argCount. Values are either a DataExpression or are a StructureVector of the form [expression,varnames...]. The values can never be a StructureString.
