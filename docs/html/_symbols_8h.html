◆ boundFunction

typedef void(* boundFunction) (ManyType &, mtvec &, long)

A type suitable to reference all of the language built-in functions. The first argument is the return value. The second, a vector of [function_name,args...]. The last is how many layer of recursion to allow in the computation.
